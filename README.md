## Installation

1. Clone this repo
2. Open `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select this folder

## Usage

### Paste Screenshot
1. Take screenshot (Cmd+Shift+4 on Mac)
2. Click MemeForge extension
3. Paste with Ctrl/Cmd+V
4. Add text
5. Click "Copy to Clipboard"

### Right-click Image
1. Find image on any webpage
2. Right-click â†’ "Edit in MemeForge"
3. Add text
4. Copy to clipboard

### Combine Images
1. Open MemeForge
2. Select layout (Stack, Side-by-Side, Grid)
3. Paste/drop multiple images
4. Add text
5. Copy to clipboard

## Keyboard Shortcuts

- `Ctrl/Cmd + V` - Paste image
- `Delete/Backspace` - Delete selected text
- `Ctrl/Cmd + C` - (on canvas) Copy meme
